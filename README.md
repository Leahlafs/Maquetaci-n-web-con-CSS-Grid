# 🧩 Landing Page con CSS Grid

## 📌 Descripción del Proyecto

Este proyecto consiste en la maquetación de una **landing page utilizando exclusivamente CSS Grid**, tomando como referencia un diseño específico con 8 bloques (Item 1 al Item 8).

El diseño ocupa **todo el espacio visible de la pantalla (viewport)** y respeta la distribución indicada en la guía proporcionada.

---

## 🎯 Objetivo

Aplicar correctamente **CSS Grid** para estructurar un layout complejo, utilizando unidades relativas y separando adecuadamente estructura (HTML) y estilos (CSS).
